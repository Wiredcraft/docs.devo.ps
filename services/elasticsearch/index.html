ElasticSearch Official documentation

Commands

start
Start ElasticSearch if stopped
stop
Stop ElasticSearch if started
reload
Reload ElasticSearch, reload the configuration and perform a graceful restart
restart
Restart ElasticSearch, reload the configuration (but kills existing connection)

Configuration

bind_host string bind_host
The IP address to bind to. Default: 127.0.0.1
group string group
Running group for the elasticsearch process Default: elasticsearch
heap_size string heap_size
Heap size for the elasticsearch process (unit as 256m 1g) Default: 256m
http_port integer http_port
Listening port for HTTP traffic Default: 9200
plugins array plugins
List of extra plugins to install Default:
transport_port integer transport_port
Listening port for inter-node traffic (cluster) Default: 9300
user string user
Running user for the elasticsearch process Default: elasticsearch
