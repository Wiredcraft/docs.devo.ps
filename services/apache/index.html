Apache Official documentation

Commands

start
Start Apache if stopped
stop
Stop Apache if started
reload
Reload Apache, reload the configuration and perform a graceful restart
restart
Restart Apache, reload the configuration (but kills existing connection)

Notes

For better user experience you may prefer the use of the reload task instead of restart.

Configuration

event
event.maxclients integer maxclients
Maximum number of simultaneous client connections Default: 150
event.maxrequestsperchild integer maxrequestsperchild
Maximum number of requests a server process serves.
event.maxsparethreads integer maxsparethreads
Maximum number of worker threads which are kept spare. Default: 75
event.minsparethreads integer minsparethreads
Minimum number of worker threads which are kept spare, Default: 25
event.startservers integer startservers
Initial number of server processes to start. Default: 5
event.threadlimit integer threadlimit
ThreadsPerChild can be changed to this maximum value during a graceful restart. ThreadLimit can only be changed by stopping and starting Apache. Default: 64
event.threadsperchild integer threadsperchild
Constant number of worker threads in each server process. Default: 25
group string group
Apache running group. Default: www-data
keepalive string keepalive
Whether or not to allow persistent connections. Default: true
keepalivetimout integer keepalivetimout
Number of seconds to wait for the next request from the same client on the same connection. Default: 5
maxkeepaliverequests integer maxkeepaliverequests
The maximum number of requests to allow during a persistent connection. Default: 100
prefork
prefork.maxclients integer maxclients
Maximum number of server processes allowed to start. Default: 150
prefork.maxrequestsperchild integer maxrequestsperchild
Maximum number of requests a server process serves. Default: 4000
prefork.maxspareservers integer maxspareservers
Maximum number of server processes which are kept spare. Default: 10
prefork.minspareservers integer minspareservers
Minimum number of server processes which are kept spare. Default: 2
prefork.startservers integer startservers
Number of server processes to start. Default: 5
timeout integer timeout
The number of seconds before receives and sends time out. Default: 300
user string user
Apache running user. Default: www-data
worker
worker.maxclients integer maxclients
Maximum number of simultaneous client connections Default: 150
worker.maxrequestsperchild integer maxrequestsperchild
Maximum number of requests a server process serves.
worker.maxsparethreads integer maxsparethreads
Maximum number of worker threads which are kept spare. Default: 75
worker.minsparethreads integer minsparethreads
Minimum number of worker threads which are kept spare, Default: 25
worker.startservers integer startservers
Initial number of server processes to start. Default: 5
worker.threadlimit integer threadlimit
ThreadsPerChild can be changed to this maximum value during a graceful restart. ThreadLimit can only be changed by stopping and starting Apache. Default: 64
worker.threadsperchild integer threadsperchild
Constant number of worker threads in each server process. Default: 25
