Official documentation

Tasks

Name Options Description
start - Start Apache if stopped
stop - Stop Apache if started
reload - Reload Apache, reload the configuration and perform a graceful restart
restart - Restart Apache, reload the configuration (but kills existing connection)

Notes

For better user experience you may prefer the use of the reload task instead of restart.
