Varnish

Varnish is a web application accelerator. You install it in front of your web application and it will speed it up significantly.

Example

services:
  varnish: '*'
configuration:
  varnish:
    listen_port: 80
    listen_address: 0.0.0.0
    storage_type: malloc
    storage_size: 512M

Install Varnish on the node, making it listen on all interfaces (0.0.0.0) and on TCP/80. The cache is set to be held in RAM (Malloc) for a maximum size of 512MB.

Configuration

  • admin_listen_address

    Telnet admin interface listen address, usually only bind localhost IP address for the admin interface.

    Type Default Required
    string 127.0.0.1 false

  • admin_listen_port

    Telnet admin interface listen port.

    Type Default Required
    integer 6082 false

  • listen_address

    Blank address means all IPv4 and IPv6 interfaces, otherwise specify a host name, an IPv4 dotted quad.

    Type Default Required
    string false

  • listen_port

    Listen port.

    Type Default Required
    integer 6081 false

  • max_threads

    The maximum number of worker threads to start.

    Type Default Required
    integer 1000 false

  • min_threads

    The minimum number of worker threads to start.

    Type Default Required
    integer 1 false

  • secret_file

    File containing administration secret.

    Type Default Required
    string /etc/varnish/secret false

  • storage_file

    Storage file, only used if storage_type is file.

    Type Default Required
    string /var/lib/varnish/$INSTANCE/varnish_storage.bin false

  • storage_size

    Cache size: in bytes, optionally using k / M / G / T suffix.

    Type Default Required
    string 256M false

  • storage_type

    Store objects either in memory (malloc), on in a file (file).

    Type Default Required
    string malloc false

  • thread_timeout

    Idle timeout for worker threads.

    Type Default Required
    integer 120 false

  • ttl

    Default TTL used when the backend does not specify one.

    Type Default Required
    integer 120 false

  • vcl_conf

    Main configuration file.

    Type Default Required
    string /etc/varnish/default.vcl false
