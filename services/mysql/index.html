Configuration

Variable Default Description
integer auto_increment_increment 1 Auto increment.
integer auto_increment_offset 1 Auto increment offset.
string base_dir /usr Base folder.
string bind_address 127.0.0.1 Binding IP address.
string data_dir /var/lib/mysql Database files path.
string innodb.innodb_adaptive_flushing true Specifies whether to dynamically adjust the rate of flushing dirty pages in the InnoDB buffer pool based on the workload.
string innodb.innodb_additional_mem_pool_size 8M The size in bytes of a memory pool InnoDB uses to store data dictionary information and other internal data structures.
string innodb.innodb_buffer_pool_size 128M The size in bytes of the memory buffer InnoDB uses to cache data and indexes of its tables.
string innodb.innodb_data_file_path ibdata1:10M:autoextend:max:500M The paths to individual data files and their sizes.
integer innodb.innodb_flush_log_at_trx_commit 1 If 0 the log buffer is written out to the log file once per second. If 1 the log buffer is written out to the log file at each transaction commit and the flush to disk operation is performed on the log file. If 2 the log buffer is written out to the file at each commit but the flush to disk operation is not performed on it.
string innodb.innodb_flush_method O_DIRECT Flush method for InnoDB, either of fdatasync, O_DSYNC or O_DIRECT
string innodb.innodb_log_buffer_size 8M The size in bytes of the buffer that InnoDB uses to write to the log files on disk.
string innodb.innodb_log_file_size 5M The size in bytes of each log file in a log group.
string logs.log_error /var/log/mysql/mysql.log The location of the error log.
string logs.log_queries_not_using_indexes false If you are using this option with the slow query log enabled then queries that are expected to retrieve all rows are logged.
number logs.long_query_time 1 Increment slow_queries variable and log if a query takes longer than this many seconds.
integer logs.slow_query_log 1 Whether the slow query log is enabled.
string logs.slow_query_log_file /var/log/mysql/slow.log The name of the slow query log file.
integer old_passwords 0 Allow old (non-secure) passwords.
string pid_file /var/run/mysqld/mysqld.pid Pid file.
integer port 3306 Connection port.
string query_cache.query_cache_limit 2M Do not cache results that are larger than this number of bytes.
string query_cache.query_cache_size 64M The amount of memory allocated for caching query results.
integer replication.expire_logs_days 10 The number of days for automatic binary log file removal.
string replication.log_bin mysql-bin Enable binary logging. The option value is the basename for the log sequence.
string replication.log_bin_trust_function_creators false Controls whether stored function creators can be trusted not to create stored functions that will cause unsafe events to be written to the binary log.
string replication.log_slave_updates true This option tells the slave to log the updates performed by its SQL thread to its own binary log.
string replication.max_binlog_size 100M If a write to the binary log causes the current log file size to exceed the value of this variable then the server rotates the binary logs.
string replication.relay_log relay-bin The basename for the relay log.
integer replication.server_id 10 Is used in replication to enable master and slave servers to identify themselves uniquely.
string replication.skip_slave_start true Tells the slave server not to start the slave threads when the server starts.
integer replication.sync_binlog 1 If the value of this variable is greater than 0 then the MySQL server synchronizes its binary log to disk (using fdatasync()) after every sync_binlog writes to the binary log.
string socket /var/run/mysqld/mysqld.sock Mysql listening socket.
integer tunable.back_log 128 The number of outstanding connection requests MySQL can have. In other words this value is the size of the listen queue for incoming TCP/IP connections.
string tunable.key_buffer 256M Size of the buffer used for index blocks. The key buffer is also known as the key cache.
string tunable.max_allowed_packet 16M The maximum size of one packet or any generated/intermediate string.
integer tunable.max_connections 256 The maximum permitted number of simultaneous client connections.
string tunable.max_heap_table_size 64M Maximum size to which user-created MEMORY tables are permitted to grow.
integer tunable.net_read_timeout 30 The number of seconds to wait for more data from a connection before aborting the read.
integer tunable.net_write_timeout 30 The number of seconds to wait for a block to be written to a connection before aborting the write.
integer tunable.table_open_cache 1024 The number of open tables for all threads.
integer tunable.thread_cache_size 32 Number of threads the server should cache for reuse.
string tunable.thread_stack 256K The stack size for each thread.
integer tunable.wait_timeout 180 The number of seconds the server waits for activity on a noninteractive connection before closing it.
