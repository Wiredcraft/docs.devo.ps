Git

Commands

devops git update
Clone or update a git repository

Options

Key Description
string dest Folder where repository is cloned on the node
string repo URL of the git repository (http or ssh)
string version Optional Version of the repository to checkout (HEAD, SHA-1, branch or tag name) Default: HEAD
devops git config
Set git user configuration either globally or for a repo defined by its path on the system

Options

Key Description
string email Email address of the user used to perform the commits
string path Optional Path of the repo on the host where to set the git configuration, if no `path` provided perform a global setup
string user Full name of the user that will perform commits
devops git deploy
Commit and push the content of a folder to a remote repository

Options

Key Description
string branch Remote branch to push the source content to (will be removed)
string repo Local folder where the repository is located
string source Source folder within the repo where the content to push is
