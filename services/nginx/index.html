Commands

start
Start MongoDB if stopped
stop
Stop MongoDB if started
reload
Reload MongoDB, reload the configuration and perform a graceful restart
restart
Restart MongoDB, reload the configuration (but kills existing connection)
vhost add
Defines a HTTP virtual host in Nginx config

Options

string aliases Optional
space separated list of domain name aliases
string domain
domain name
integer port
listening port
array routes
list of route objects

route Object

string custom Optional
Custom inline nginx config to include within the route (e.g. auth, custom timeout)
string static Optional, Default: root
For type static only, define how to consider the source folder - alias or root Valid values: Either of alias / root
string to
Either an upstream name, or a service / url, or a path
string type
The type of handler for that route Valid values: Either of proxy / fastcgi / websocket / uwsgi / static
string uri
Any string / regex that nginx understand as a `location` Valid values: Any string including regex
array upstreams Optional
list of upstream objects

upstream Object

array backends
List of backends associated with the upstream
string name
Name of the upstream - it must be unique on the entire node
string webroot Optional
subfolder to serve data from based on the root /var/www/_domain_

Configuration

Number of concurrent http connection handled per nginx process

string http.error_log Optional, Default: /var/log/nginx/error.log
Global error log file path
string http.gzip Optional, Default: on
Enable gzip compression
string http.gzip_disable Optional, Default: msie6
Space separated list of browser to disable gzip compression for
integer http.keepalive_timeout Optional, Default: 65
Keepalive timeout (sec) - 0 to disable
string http.sendfile Optional, Default: on
Enable sendfile
string http.tcp_nodelay Optional, Default: on
Enable tcp_nodelay
string http.tcp_nopush Optional, Default: on
Enable tcp_nopush
integer http.types_hash_max_size Optional, Default: 2048
Maximum size of hash tables
string pid Optional, Default: /var/run/nginx.pid
Pid file path
string user Optional, Default: www-data
Nginx running user.
integer worker_processes Optional, Default: 4
Number of Nginx processes
