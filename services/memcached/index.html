Official documentation

Tasks

Name Options Description
start - Start Memcached if stopped
stop - Stop Memcached if started
reload - Reload Memcached, reload the configuration and perform a graceful restart
restart - Restart Memcached, reload the configuration (but kills existing connection)

Configuration

Variable Default Description
integer connection 1024 Maximumum number of concurrent inbound connections.
string enable yes Enable or disable memcached service.
string listen 127.0.0.1 Binding IP address.
string logfile /var/log/memcached.log Log file path
integer memory 64 Amount of memory allocated in MB.
integer port 11211 Connection port.
string user memcache Memcached running user.
