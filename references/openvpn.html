OpenVPN

SSL/TLS based user-space VPN. Supports Linux, Solaris, OpenBSD, FreeBSD, NetBSD, Mac OS X, and Windows 2000/XP.

Example

services:
  openvpn: '*'
configuration:
  openvpn:
    port: 6543
    proto: 'tcp'
    max_client: 10
    key:
      key_size: 2048
      email: me@example.com

Install OpenVPN on the node, configure the service so it listen on TCP/6543 (instead of regular UDP/1194), limit the number of concurrent clients to 10 and set the server's key to be 2048bits with a custom email in the certificate.

Note that the default values of the configuration will satisfy most of the use and those customization are more for advanced users.

The management of OpenVPN clients is handled by dedicated tasks.

Configuration

  • port

    Listening port

    Type Default Required
    integer 1194 false

  • proto

    Protocol

    Type Default Required
    string udp false

  • dev

    Device type

    Type Default Required
    string tun false

  • network

    Network IP range for clients

    Type Default Required
    string 10.8.0.0 false

  • netmask

    Netmask to apply on IP range for clients

    Type Default Required
    string 255.255.255.0 false

  • dns1

    DNS to propagate to the clients (default to OpenDNS)

    Type Default Required
    string 208.67.222.222 false

  • dns2

    DNS to propagate to the clients (default to OpenDNS)

    Type Default Required
    string 208.67.220.220 false

  • cipher

    cipher

    Type Default Required
    string BF-CBC false

  • max_client

    Maximum number of simultaneous clients

    Type Default Required
    integer 100 false

  • user

    Running user of openvpn (must exist)

    Type Default Required
    string nobody false

  • group

    Running group of openvpn (must exist)

    Type Default Required
    string nogroup false

  • key

    • key_size (key.key_size)

      Increase key size if you are paranoid; it will slow down TLS negociation

      Type Default Required
      integer 1024 false

    • ca_expire (key.ca_expire)

      CA validity period (in days)

      Type Default Required
      integer 3650 false

    • key_expire (key.key_expire)

      Key validity period (in days)

      Type Default Required
      integer 3650 false

    • country (key.country)

      Key country

      Type Default Required
      string US false

    • province (key.province)

      Key province

      Type Default Required
      string CA false

    • city (key.city)

      Key city

      Type Default Required
      string SanFrancisco false

    • org (key.org)

      Key Organization

      Type Default Required
      string Fort-Funston false

    • email (key.email)

      Key email

      Type Default Required
      string admin@server false

    • common_name (key.common_name)

      Key Common Name (CN)

      Type Default Required
      string server false

    • name (key.name)

      Key name

      Type Default Required
      string server false

    • org_unit (key.org_unit)

      Key Organization Unit (OU)

      Type Default Required
      string server false
