PHP

PHP is a popular general-purpose scripting language that is especially suited to web development.

Fast, flexible and pragmatic, PHP powers everything from your blog to the most popular websites in the world.

Example

services:
  php: '*'
configuration:
  php:
    memory_limit: 64M
    post_max_size: 32M
    upload_max_filesize: 32M
    apc:
      shm_size: 256M

Install PHP on the node, setting the max limit of a php process to 64MB, allowing 32MB files upload, and giving 256MB of room to APC cache.

    string Off false

    Type Default Required
    integer 60 false

  • max_input_time

    Maximum amount of time each script may spend parsing request data.

    Type Default Required
    integer 120 false

  • memory_limit

    Maximum amount of memory a script may consume.

    Type Default Required
    string 128M false

  • error_reporting

    Define which errors, warnings and notices to report.

    Type Default Required
    string E_ALL & ~E_DEPRECATED false

  • display_errors

    Define whether PHP will output errors, notices and warnings.

    Type Default Required
    string Off false

  • log_errors

    Define whether PHP should log errors to locations such as a server-specific log, STDERR, or a location specified by the error_log.

    Type Default Required
    string On false

    Log errors to specified file (or syslog).

    Type Default Required
    string syslog false

  • post_max_size

    Maximum size of POST data that PHP will accept.

    Type Default Required
    string 16M false

  • upload_max_filesize

    Maximum allowed size for uploaded files.

    Type Default Required
    string 16M false

      Type Default Required
      integer 1 false

    • save_handler (session.save_handler)

      Handler used to store/retrieve session data.

      Type Default Required
      string files false

      Folder PATH where sessions will be stored (if save_habdler is file).

      Type Default Required

      Enable / disable APC (0 - disabled).

      integer 1 false

      Type Default Required

      Type Default Required

      The number of seconds a user cache entry is allowed to idle in a slot in case this cache entry slot is needed by another entry.

          List (comma separated) of ipv4 addresses of FastCGI clients which are allowed to connect.

          Type Default Required

          Choose how the process manager will control the number of child processes. (static, dynamic, ondemand)

          Type Default Required
