# Units in the configuration

The various size or time configuration items can be expressed in the following units.

## Memory units

  kB = kilobytes
  MB = megabytes
  GB = gigabytes

## Time units

  ms  = milliseconds
  s   = seconds
  min = minutes
  h   = hours
  d   = days
